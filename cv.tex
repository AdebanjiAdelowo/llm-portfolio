\section{Selected Projects}

\noindent
\textbf{\href{https://github.com/adebanjiadelowo/llm-portfolio/tree/main/01-financial-sentiment-distillation}{Financial Sentiment Distillation}} ---
Compressed FinBERT by 62\% (110M $\to$ 88M parameters) via Taylor-gradient attention-head pruning and knowledge distillation, improving accuracy from 84.5\% to \textbf{96.9\%} on the financial\_phrasebank benchmark.
\textit{Stack: PyTorch, HuggingFace Transformers, scikit-learn}

\textbf{\href{https://github.com/adebanjiadelowo/llm-portfolio/tree/main/02-enterprise-nl2sql}{Enterprise NL2SQL Pipeline}} ---
Two-stage LLM pipeline for natural language to SQL over an 8-table enterprise database. A lightweight table-selector cuts prompt token cost by \textbf{$\sim$60\%}; a complexity router auto-escalates multi-table queries to a stronger model. Includes retry logic and SQL injection guard.
\textit{Stack: OpenAI API (GPT-4o-mini), Python, pandas}

\textbf{\href{https://github.com/adebanjiadelowo/llm-portfolio/tree/main/03-bank-customer-risk-engine}{Bank Customer Risk Engine}} ---
Embedding-powered risk assessment system encoding customer profiles into 384-dim sentence-transformer vectors. A multi-output model simultaneously predicts loan approval probability, default risk, and adjusted interest rate; cosine-similarity search enables nearest-neighbour customer profiling.
\textit{Stack: sentence-transformers, XGBoost, scikit-learn, UMAP}

\textbf{\href{https://github.com/AdebanjiAdelowo/generative-ai-portfolio/blob/main/End-to-End-AI-Applications/End-to-end-Medical-Chatbot-Generative-AI}{RAG Medical Chatbot}} --- Built medical Q\&A system using RAG; integrated Pinecone for semantic search and context-aware responses. \textit{Stack: OpenAI, Pinecone, LangChain, Flask}
